# ntc-jredis
ntc-jredis is module redis java client.  

## Maven
```Xml
<dependency>
    <groupId>com.streetcodevn</groupId>
    <artifactId>ntc-jredis</artifactId>
    <version>1.0.0</version>
</dependency>
```
